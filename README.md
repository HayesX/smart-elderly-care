# 智慧助老综合平台

> 如果科技有温度，那它应该从照顾父母开始。

一款专为老年人设计的一站式智慧养老 Web 应用，让独居和空巢老人也能享受科技便利。

**在线预览**：https://hayesx.github.io/smart-elderly-care/

## 功能概览

### 老人端

| 功能 | 说明 |
|------|------|
| 拍照识药 | 拍照识别药品说明书，大字提醒用药，再也不会吃错药 |
| 用药计划 | 每日用药提醒，大字版用药清单 |
| AI 陪伴聊天 | 随时在线的聊天伙伴，支持语音输入与播报 |
| 一键叫车 | 大字版界面语音叫车，无需下载 APP |
| 天气出行 | 大字天气卡片 + 出行建议 |
| SOS 紧急呼叫 | 超大按钮一键求助，自动通知子女 |

### 子女端

| 功能 | 说明 |
|------|------|
| 用药看板 | 远程查看父母今日用药情况 |
| 安全守护 | 异常行为实时预警，活动状态时间线 |
| 爸妈心情 | 心情指数曲线 + 聊天关键词 |
| 报告导出 | 周/月健康报告一键导出 |

## 设计理念

- **大字优先**：正文 ≥ 18px，标题 ≥ 24px，关键信息 ≥ 28px
- **按钮够大**：主操作按钮 ≥ 48px，SOS 按钮 ≥ 64px
- **操作够简**：语音输入优先，减少打字和层级跳转
- **温暖配色**：暖橙主色 `#e8734a`，柔和易读

## 技术栈

- React 18 + TypeScript
- React Router v6（HashRouter，兼容 GitHub Pages）
- Vite 5（`base: './'` 相对路径构建）
- Web Speech API（语音识别 + 语音播报）
- 纯前端 Mock 数据，无需后端即可体验

## 本地运行

```bash
npm install
npm run dev
```

浏览器访问 http://localhost:5173/

## 构建部署

```bash
npm run build      # 构建到 dist/
npm run preview    # 本地预览生产构建
```
